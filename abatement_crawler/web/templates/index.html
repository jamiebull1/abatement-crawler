🌱 Abatement Crawler

Automatically discover, extract, and organise data on carbon abatement measures from web sources.

What is this tool?

The Abatement Crawler is a web crawler that searches the internet for documents describing carbon abatement measures — technologies and practices for reducing greenhouse gas emissions. It uses AI to extract structured cost and carbon-performance data, then stores everything in a searchable local database.

Core capabilities:

  • Multi-source search — DuckDuckGo (free, no key needed), SerpAPI, Google Custom Search Engine, or Bing
  • Multi-format ingestion — HTML, PDF, Excel, DOCX, JSON
  • AI-powered extraction — Anthropic Claude extracts structured records
  • Quality scoring — ranks records by evidence quality
  • Data normalisation — currency, geography and units standardisation
  • Flexible export — JSONL, CSV, Parquet or Markdown report
Getting started
  1. Set your API keys — open Configuration and enter your Anthropic API key (for LLM extraction). A Search API key is only needed when using Google CSE, Bing, or SerpAPI instead of the default DuckDuckGo.
  2. Define your scope — specify the industry, process, geographies and abatement types you want to collect data on.
  3. Run a crawl — go to the Crawl page and start a crawl in search mode (automatic query building) or seed mode (from a list of URLs you provide).
  4. Explore results — browse and filter extracted records on the Results page, or export them from the CLI.
Crawl modes
Search mode default
Automatically builds search queries from your scope configuration and uses a search API to find relevant documents. The crawler then follows links in a priority queue, preferring high-quality domains.
Seed mode
Starts from a list of URLs you provide. Useful when you already have specific reports or data sources you want to extract data from.
API keys required
Key Purpose Where to obtain
Anthropic API key LLM extraction via Claude console.anthropic.com
Search API key Web search (Google CSE / Bing / SerpAPI — not needed for DuckDuckGo) Google CSE · Bing · serpapi.com
Keys can also be supplied via the environment variables ANTHROPIC_API_KEY and SEARCH_API_KEY.
CLI usage

All functionality is also available from the command line:

# Run in search mode (default)
abatement-crawler crawl --config config/config.yaml --mode search

# Run in seed mode
abatement-crawler crawl --config config/config.yaml --mode seed \
  --seed-urls https://example.com/report.pdf

# Export results
abatement-crawler export --config config/config.yaml --format csv
abatement-crawler export --config config/config.yaml --format jsonl --min-quality 0.5

# Launch the web UI
abatement-crawler web --config config/config.yaml
